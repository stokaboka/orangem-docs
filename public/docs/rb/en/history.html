RedButton::History

While developing information systems working with databases, we came to the conclusion that at the moment Web-applications are, perhaps, the best solution.

Experience of development in systems

  • Oracle Developer Forms/Reports 6 -6i
  • Oracle JDeveloper
led to the idea of building a tool for rapid development of a data access system based on
  • Linux (server part)
  • NodeJS (server part)
  • MySQL, PostgreSQL, MongoDB (data bases)
  • HTML5 (client part)

We used only open source tools (Open Source).

  • Impress Application Server as an application server and a web server
  • AngularJS as a framework for the client part

Use of web-technologies allows

  • simplify and make cheaper the administration of the information system
  • the ability to access corporate data from any device, including mobile and thin clients that support modern web browsers: Chrome, FireFox, Edge, Opera, Safari, etc

We closely follow the new technologies and are now actively studying and developing new tools using VueJS. The migration from AngularJS to VueJS is planned
